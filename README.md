## 📌 Overview
This project analyzes retail sales data using SQL to generate meaningful business insights.

## 🛠 Tools Used
- SQL
- Retail Sales Dataset

## 🔍 Analysis Performed
- Total revenue and profit calculation
- Category-wise sales analysis
- Top-selling products identification
- Regional performance analysis
- Monthly sales trend analysis

## 📊 Key Insights
- Identified top-performing products and categories
- Analyzed revenue distribution across regions
- Observed sales trends over time

## 📁 Files
- dataset.csv → Dataset used for analysis
- sql_queries.sql → Contains SQL queries

## 🚀 Conclusion
This project demonstrates how SQL can be used to analyze business data and support decision-making.
